## 2. $0 채널별 집행 계획

### X (Twitter) Japan — 주력 채널
- **전략:** 運命鑑定 / 四柱推命 / AI占い 해시태그 공략
- **목표:** 팔로워 0 → 200 (론칭 전 4주), 론칭 주 도달 2,000+
- **전술:**
  - 트렌딩 운세 키워드 리트윗 + 코멘트 (하루 3~5건)
  - 무료 사주 일부 공개 스레드 (주 2회)
  - "あなたの干支は？" 투표 기능 활용 (참여 유도)
  - AI占い 관련 계정 팔로우 → 유기적 팔로우백 유도

### LINE OA — 전환 채널
- **전략:** X에서 클릭 → LINE 친구 추가 → 웰컴 메시지 → 결제
- **전술:**
  - 프로필 링크에 LINE OA URL 고정
  - 트윗 1개에 반드시 "LINEで占う→" CTA 포함
  - 리치메뉴: 四柱推命 / 相性占い / 今月の運勢 3버튼 구성

### note.jp — SEO + 신뢰도 채널
- **전략:** 四柱推命解説 롱폼 콘텐츠로 검색 유입
- **전술:**
  - 주 1회 1,500자+ 운세 해설 (무료 공개)
  - "AIが見た2026年下半期の運勢" 시리즈
  - 말미에 MEI LINE 링크 삽입
  - 인기 note 작가 コメント 적극 참여

### TikTok Japan — 바이럴 채널 (2주차~)
- **전략:** 60초 以内 운세 동영상 (텍스트+BGM)
- **전술:**
  - "あなたの四柱推命を調べてみた" 포맷
  - 생년월일 입력 → 결과 화면 캡처 공유 유도
  - 트렌딩 BGM 사용 (著作権フリー)

## 4. PMF 측정 지표

### 핵심 KPI
| 지표 | W+1 목표 | W+4 목표 | 측정 방법 |
|------|---------|---------|---------|
| 결제 건수 | 5건 | 30건 | LINE Pay 영수증 |
| LINE 친구 수 | 50명 | 200명 | LINE OA 관리자 |
| X 팔로워 | 150명 | 500명 | X Analytics |
| LIFF 접속 수 | 200 PV | 1,000 PV | Cloudflare Analytics |
| 결제 전환율 | 2.5% | 3% | 결제/LIFF 접속 |

### PMF 판단 기준 (결제 5건 달성 조건)
1. LINE 친구 추가 수 ≥ 50
2. LIFF 접속 후 결제 페이지 도달율 ≥ 10%
3. 결제 완료 후 LINE 차단 없이 유지 ≥ 80%
4. 재결제 (相性占い 등 2번째) ≥ 1건
