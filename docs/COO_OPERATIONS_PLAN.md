## 3. KPI 목표치

| 지표 | W4 목표 | W8 목표 | 측정 도구 |
|------|--------|--------|---------|
| **월 유료 결제 건수** | **10건** | **33건** | Stripe 대시보드 |
| 월 순이익 | $28 | $93 | CFO 시나리오 (§1) |
| 무료 체험 건수 | 200건 | 660건 | CF Workers KV 집계 |
| LINE 친구 수 | 50명 | 200명 | LINE OA Manager |
| 사주 API 응답 성공률 | 99% | 99.5% | CF Workers Analytics |

**전환율 가설 (추정 — 실측 데이터 없음):**
- **무료 체험 → 유료 전환 5% 가정.** 이는 업계 일반치 기반 추정이며 검증된 수치가 아니다.
- 역산: 유료 10건 = 무료 200건 필요 / 유료 33건 = 무료 660건 필요 (5% 가정 하 산술 계산).
- **D-day 후 2주 시점에 실측 전환율로 가설 교체.** 실측 <2%면 가격·체험 설계 재검토(CSO 소집), >8%면 무료 체험 노출 확대.

**KPI 주차 기준:** W4/W8은 기준일(06/11)부터 계산하되, **D-day가 W3을 넘기면 KPI 주차는 D-day 기준으로 순연** (Stripe 심사는 통제 불가 변수 — 지연이 KPI 미달로 집계되면 안 됨).
